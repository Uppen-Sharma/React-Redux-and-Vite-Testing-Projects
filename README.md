# React Projects & Learnings Collection

This repository is a collection of various React sub-projects, each focusing on different aspects of the React ecosystem. This serves as a continuous learning environment to practice state management, tooling, testing, and modern React patterns.

## 🚀 Projects Overview

Here is a summary of the projects included in this repository and the key learnings from each:

### 1. `Vite-React` & `vite-project`

- **Focus**: Modern Development Environment
- **Learnings**: Setting up and running React applications using [Vite](https://vitejs.dev/) for significantly faster hot-module replacement (HMR) and optimized build processes compared to Create React App.

### 2. `context-app` & `context-api-themes`

- **Focus**: Global State Management (Context API)
- **Learnings**:
  - Utilizing the React Context API (`createContext`, `useContext`) to avoid prop drilling.
  - Building a robust, globally accessible theme-switcher (Dark/Light mode) in `context-api-themes` to handle UI styling dynamically across the entire component tree.

### 3. `redux-react`

- **Focus**: Predictable State Management
- **Learnings**:
  - Integrating Redux into a React application using `react-redux` and modern Redux Toolkit (RTK) practices.
  - Managing complex application state, structuring reducers, and dealing with structured unidirectional data flow.

### 4. `Testing`

- **Focus**: Quality Assurance & Component Reliability
- **Learnings**: Writing isolated unit and integration tests for React components, ensuring that features function as expected and catching bugs early using standard testing frameworks (e.g., Jest, React Testing Library).

### 5. `bug-blaster`

- **Focus**: Comprehensive Application Building
- **Learnings**: A hands-on project that ties together multiple concepts (state management, component composition, side effects) to build a functional tool for tracking/squashing bugs.

---

_This repository is continuously updated as new patterns and tools are explored within the React and Redux ecosystems._
